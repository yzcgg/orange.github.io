第二章

GSM

全球移动通信系统Global System for Mobile communications, 主要服务陆地, 是第二代移动通信系统2G.

GSM的网络结构

  • 移动台 MS

    简单理解类似手机一样的东西，包括移动台物理设备(ME)和智慧部件SIM卡两部分，类型有车载台、便携台以及手持台等。

    • 移动终端ME
      • 话音加密
      • 信道编码
      • 信息加密
      • 信息的调制和解调
      • 信息发射和接收
    • 客户识别卡 SIM
      • 存储了用户的各种信息, 包括国际移动用户识别码IMSI号, PIN码等, 它与网络数据库中存储的信息一致

  • 无线基站子系统 BSS Base Station Subsystem

    由MSC控制, 与MS进行通信的系统设备. 主要完成无线发送, 接收和无线资源管理.
    BSC: 基站控制器. BTS:基站收发信机.

  • 网络交换子系统 NSS Network service subsystem

  • 操作维护子系统 OMC

移动终端ME
E 是 equipment
话音编码/信源编码(64k b/s), 信道编码, 信息加密, 信息的调制和解调, 信息发射和接受

位置区信息 不能太小也不能太大
