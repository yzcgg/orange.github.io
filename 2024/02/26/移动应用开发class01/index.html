内容

  • 概述
  • 基础通信网络
  • 终端
  • Android 开发
  • ios 开发

要求

  • 平时作业,手机编程作业
  • 大作业: 设计实习一个移动互联网的应用,题目不限

CH_1 概述

移动互联网的定义

三要素:

  • 网络: GPRS, 3G, 4G(全ip化), 5G, (RFID, WiFi,卫星)等移动通信网络实现终端接入互联网
  • 终端: 手机, 笔记本, 快递站的扫码设备…
  • 服务: 社交, 支付, 搜索, 游戏…

移动互联网的演进

  1. 移动增值网: 增值业务
  2. 独立wap网站: 网站独立于运营商, 直接面向消费者. WAP网关.
  3. 移动互联网
  4. 宽带无线互联网

可以让服务端以为你的手机是一个PC机, 而不是一个手机.

  • UA : user agent
  • APN : APN指一种网络接入技术，是通过手机上网时必须配置的一个参数，它决定了手机通过哪种接入方式来访问网络. access point name.

移动互联网特点

  1. 便携 + 位置化
    手机是怎么定位的?

    基站:随时随地广播信息.三角定位:多个基站,可以确定某个终端的位置.

共享单车定位?

GPRS或者通过单车连接手机定位手机

  1. 即时性 + 永远在线
  2. 交互性

    二维码, 音视频, 重力感应, NFC…
    手机的NFC的卡模式 or 全模式: 充当NFC的读卡器 and 标签. 还是只能充当标签.

  3. 时间碎片化
  4. 终端能耗

移动互联网与互联网的关系

签名:
通过加密和解密完成的. U盾.

物联网

愿景: 高级军用车辆 – 智能轮胎压力控制器.

移动互联网的体系架构

云: 业务与应用
管道: 接入, 地址分配, 漫游
终端: 硬件, 操作系统, 应用软件

ip要保证在很大一个范围内不能变, 比如整个北京城区.

作业

  1. APN
  2. 查看手机IP地址, 使用流量和wifi, IP的归属地是那里.
  3. 用手机和电脑分别访问: wap.qq.com wap.baidu.com 等手机网站, 有啥不同.
  4. 两个手机之间传输信息, 必须联网吗?

    蓝牙. 二维码.
    wifi两种模式 : dcf 和 pcf
